# Shared ToDo List

## 서비스 소개

Shared Todo List는 개인과 공동 사용을 모두 지원하는 할 일 관리 서비스입니다.
회원가입 후 혼자 사용할 수 있으며, 상대를 초대해 하나의 투두 리스트를 함께 공유할 수 있습니다.
개인 일정부터 협업이 필요한 할 일까지 효율적으로 관리할 수 있습니다.
